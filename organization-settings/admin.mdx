# **Admin Features & Management**

Shimmer provides comprehensive administrative capabilities for organization admins to manage users, content organization, platform settings, and analytics. This guide covers all admin features and best practices for platform management.

## **User Management**

![68d1ca404832c9d1e5ef4875 Shimmer Settings User Management Pn](/images/68d1ca404832c9d1e5ef4875_shimmer-settings-user-management.png)

### **Invitation System**

**Creating User Invitations**

1. Navigate to Settings \> User Management
2. Click "Add User" button
3. Invitation link automatically generated and copied to clipboard
4. Share the copied link with new users
5. Track pending invitations in the management interface

## **Category Management**

![68d1c8e2d854a554c50efb1a Shimmer Settings Categroy Management Pn](/images/68d1c8e2d854a554c50efb1a_shimmer-settings-categroy-management.png)

### **Category System Overview**

- Organize content by category with color-coding
- Required field for all content uploads
- Filterable in content library

### **Creating and Managing Categories**

**Adding New Categories**

1. Navigate to Settings \> Categories
2. Click "Add Category" button
3. Enter category name and description
4. Select color for visual identification
5. Save category for immediate availability

**Category Best Practices**

- Use clear, descriptive category names
- Maintain consistent color coding system
- Limit number of categories for simplicity
- Regular review and cleanup of unused categories

## **Tag Management**

![68d1ca5ce2891053677997d1 Shimmer Settings Tag Management Pn](/images/68d1ca5ce2891053677997d1_shimmer-settings-tag-management.png)

### **Creating and Managing Tags**

**Tag Creation**

1. Navigate to Settings \> Tags
2. Add new tags with clear descriptions
3. Optionally: You can add new tags while uploading content

**Tag Best Practices**

- Use specific, descriptive tags
- Maintain tag consistency across content
- Regular tag audit and cleanup

### **Slack Integration Management**

![68d1ca8225203de869150521 Shimmer Settings Slack Integration Pn](/images/68d1ca8225203de869150521_shimmer-settings-slack-integration.png)

**Connecting Your Slack**

1. Click the "Add to Slack" button
2. Select the Slack channel where you'd like Shimmer to post to
3. Click "Allow"
4. Click "Test" to test the Slack channel connect (a test message will be posted to the Slack channel you connected)
